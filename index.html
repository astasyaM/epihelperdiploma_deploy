О приложении

ЭпиСвязь - мобильное приложение для Android. Помогает пациенту фиксировать приступы, получать оповещения при подозрительной активности и быстро передать геолокацию доверенным контактам.

  • Мониторинг Анализ акселерометра и гироскопа в фоне, ML-модель на устройстве.
  • SOS SMS и звонок доверенным контактам с координатами.
  • Дневник Учёт приступов и отчёт PDF. Автоматическая запись + возможность ручного добавления и редактирования эпизодов.
  • Контакты Настройка каналов оповещения для каждого человека.

Установка на Android

Важно: перед установкой разрешите «Установку из неизвестных источников» для вашего браузера (Настройки → Безопасность). Без этого APK не откроется.

Нажмите «Скачать APK» внизу экрана (или в секции «Скачать»). Дождитесь окончания загрузки в панели уведомлений.

Откройте загруженный файл. Если система спросит — включите установку из неизвестных источников только для браузера, через который скачали APK.

Выдайте разрешения (геолокация, сенсоры, уведомления, SMS) и зарегистрируйтесь.

Частые вопросы

Безопасно ли скачивать APK?

Да, если файл загружен с этого официального сайта. APK подписан ключом разработчика, передаётся по HTTPS. Сверьте SHA-256 на странице с контрольной суммой файла после загрузки.

Что делать при ложном срабатывании?

Нажмите «Это ложное срабатывание» в уведомлении. Запись сохранится в дневнике с пометкой.

Сколько заряда батареи потребляет приложение?

Менее 8% в сутки при активном мониторинге. Зависит от модели телефона.

Как работает мониторинг без интернета?

Приложение сохраняет данные локально и отправляет SMS даже без интернета. При появлении соединения данные синхронизируются.

Можно ли использовать приложение с умными часами?

Пока приложение работает только с сенсорами телефона. Интеграция с часами в разработке.

Работает ли приложение без интернета?

Да. Фоновый мониторинг по сенсорам работает на устройстве. SOS через SMS и звонок не требуют постоянного интернета. Синхронизация дневника с облаком — при появлении сети.

Какие разрешения нужны и зачем?

Геолокация — координаты при SOS.
Сенсоры — мониторинг движений.
Уведомления — тревоги и напоминания.
SMS и звонок — оповещение близких.
Фоновая работа — непрерывный мониторинг.

Почему приложения нет в Google Play?

Пилотное развёртывание через сайт. Нужны фоновый мониторинг, SMS и звонки. В перспективе — RuStore или Google Play.

Как обновить версию?

Скачайте новый APK с сайта и установите поверх старой (тот же ключ подписи). Приложение может напомнить о новой версии при запуске.
